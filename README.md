simple mods:

after exporting custom map code use ctrl i paste to add your custom map

add these through ctrl i paste or add to end of custom map code
  SPEED*=1 change number to adjust speed 
  BOUNCE=0.3 change number to adjust speed lost or gained after touching wall (any number over 1 gains you speed rather than losing)
  MOUNTAIN_DIST=325 distance of mountains from spawn
  OOB_DIST=300 length between walls
  LAPS=10 laps raced
